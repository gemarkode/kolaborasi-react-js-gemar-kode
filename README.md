# Kolaborasi React JS

Pull request ke repository ini jika kamu ingin berkolaborasi.

## tools

- kami sarankan anda memakai linux ya :)

- npm minimal versi 6

- node js minimal versi 10

- text editor bebas, kami sarankan atom.io atau sublime text

- Git / https://git-scm.com

## development

Clone repository

`
https://github.com/gemarkode/kolaborasi-react-js-gemar-kode/
`

install node module

`
npm install
`

jalankan server

`npm start
`

buka http://localhost:3000/
